# tremo-support

TREMOアプリのサポートページです。

## サイトURL

**https://kubotake.github.io/tremo-support/**

- [サポートページ](https://kubotake.github.io/tremo-support/)
- [プライバシーポリシー](https://kubotake.github.io/tremo-support/privacy.html)

## ページ構成

| ファイル | 内容 |
|---|---|
| `index.html` | トップ / サポートページ（概要・FAQ・お問い合わせ） |
| `privacy.html` | プライバシーポリシー（App Store申請用） |

## GitHub Pages の公開設定

1. GitHub でこのリポジトリを開く
2. **Settings** → **Pages** を開く
3. **Source** を `Deploy from a branch` に設定
4. Branch を `main`、フォルダを `/ (root)` に設定して **Save**
5. 数分後に `https://kubotake.github.io/tremo-support/` で公開される
